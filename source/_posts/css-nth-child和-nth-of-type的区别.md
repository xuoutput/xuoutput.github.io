# 'css-:nth-child 和:nth-of-type 的区别'

说的就是

- `:nth-child` 这个就是看子, 然后又有 `if` 这个条件
- `:nth-of-type` 直接看这种类型的第几个, 都是同类型就没有 `if` 的判断了
