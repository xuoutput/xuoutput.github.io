我前面已经介绍了箭头函数，但它们非常重要，它们需要再重点介绍一下。

在 ES6/ECMAScript2015 中引入了箭头函数，自从它们引入后，它们彻底改变了 JavaScript 代码写法（和工作方式）。

在我看来，这种变化非常受欢迎，你现在很少在现代代码库中看到 function 关键字。
